# IM 即时通讯系统（C++）

一个用 C++ 写的简单即时通讯系统，主要实现了客户端和服务端的通信流程，包括登录、聊天、连接管理等基础功能。

这个项目是我从 0 到 1 独立完成的，主要用来练习网络编程和并发处理。

---

## 做了什么

- 实现了基于 TCP 的客户端/服务端通信
- 使用 select 做 IO 多路复用，支持多个连接
- 自定义通信协议，解决粘包问题
- 加了简单的 ACK 机制，保证消息能送达
- 实现了心跳检测，断线能自动清理
- 用多线程 + 队列做消息处理

---

## 项目结构

- server：服务端代码
- user：客户端代码
- IM_TEST：简单压测 / 测试程序

---

## 跑起来（很简单）

1. 打开 server，启动服务端  
2. 打开 user，启动客户端  
3. 登录后可以进行简单聊天测试  

---

## 一点说明

这个项目主要是练习用，没有做复杂的安全和优化，但核心的通信流程和并发模型是完整的。

压测大概能到几万 QPS（本地测试），算是一个比较基础的高并发尝试。
